SEMANTIC VS NON SEMANTIC ELEMENT

SEMANTIC
menjelaskan tag html secara jelas tujuannya

NON SEMANTIC
Kebalikan dari semantic


INLINE ELEMENT VS BLOCK ELEMENT HTML

INLINE ELEMENT 
tidak akan membuat line baru

BLOCK ELEMENT HTML
membuat line baru

PADDING & MARGIN
PADDING
Jarak tempi container ke dalam element

MARGIN
jarak tepi container ke luar element
